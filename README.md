### Ruokavaraston tuotteiden etusivu. Uuden tuotteen pystyy lisäämään listan alapuolelta
![pa](./Pantry11.jpg)
### Tällä sivulla voi lisätä uuden tuotteen tai poistua sivulta, jos tahtoo Cancel-napin kautta

![ds](./Pantry2.jpg)
